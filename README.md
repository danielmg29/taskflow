# TaskFlow - Adaptive Convergence Project Manager

A modern project and task management system built with the Adaptive Convergence philosophy.

## Tech Stack

- **Backend:** Django 5.x with functional architecture
- **Frontend:** Next.js 14 with App Router
- **Database:** PostgreSQL (Neon)
- **Caching:** Redis
- **Containerization:** Docker
- **Version Control:** Git & GitHub

## Philosophy

This project demonstrates the Adaptive Convergence paradigm:
- Schema-Driven Development (SDD)
- Performance-First Architecture (PFA)
- Zero-Redundancy Principle (ZRP)

Backend defines data structures once. Frontend adapts automatically.
